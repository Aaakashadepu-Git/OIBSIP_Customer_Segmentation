# OIBSIP_Customer_Segmentation
# Customer Segmentation using K-Means

## 📌 Project Overview
This project uses K-Means clustering to segment customers based on income and spending behavior.

## 📊 Dataset
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## ⚙️ Model Used
- K-Means Clustering

## 📈 Results
- Customers divided into 5 clusters
- High income & high spending customers are most valuable
